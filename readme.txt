Displays a list or dropdown of taxonomies in a sidebar widget.

== Description ==

This plugin displays a list or dropdown of taxonomies in a sidebar widget.

== Installation ==

1. Download `tg29359-taxonomy-widget.zip`, upload it to your server, expand it as `/wp-content/plugins/tg29359-taxonomy-widget/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add `tg29359 taxonomy widget` to your sidebar through the 'Appearance/Widget' menu in WordPress.
